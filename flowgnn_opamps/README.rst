==============
FlowGNN on Ckt-Bench101
==============


    Code for the experiments with the FlowGNN models on the operational amplifiers from the Ckt-Bench101 dataset.


==============
Installation
==============

**Requirements**

- CPU or NVIDIA GPU
- Linux / Windows
- Python 3.12
- PyTorch 2.4.0
- PyTorch Geometric 2.6.0

.. code-block:: python

   pip install -r requirements.txt
   pip install .
